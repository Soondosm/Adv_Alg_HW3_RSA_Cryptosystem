This program was written in VSCode on a Mac (Visual Studio does not support C++ on Mac).

hw3.cpp: The C++ file which contains the full program.
Results_ScreenShot: the screenshot of the output.

TO RUN: 
1. In terminal, cd to the directory hw3.
2. type $ g++ hw3.cpp, then press enter
3. type ./hw3, then press enter

Program will then run. Enter a test string (not case-sensitive), enter a public key e.
Program will take these inputs and output p, q, n, M, C, P, e.
